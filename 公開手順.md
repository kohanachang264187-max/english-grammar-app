# 公開手順

## 静的Webホスティング

このフォルダ内のファイルを、フォルダ構成を変えずに静的Webホスティングへ配置します。

公開後に確認する項目：

1. トップ画面に「英文法12」が表示される
2. 「今日の12問を始める」で問題が表示される
3. 選択式で正誤が表示される
4. 音声ボタンを押した際にブラウザからマイク許可が求められる
5. 学習後にページを閉じても記録が残る
6. 「記録」からJSONバックアップを出力できる
7. ホーム画面へ追加後、単独アプリのように起動できる

## ローカル確認

パソコンにPythonがある場合、このフォルダで次を実行できます。

```bash
python -m http.server 8000
```

ブラウザで`http://localhost:8000`を開きます。

ローカル確認は画面・タップ・文字回答のテスト用です。iPhoneやiPadで音声認識とホーム画面追加を使う際はHTTPS公開版を使用してください。
