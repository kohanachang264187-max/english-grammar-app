# 英文法12 完成版

高校1年生から高校2年生終了までに、地方国立大学受験に必要な英文法を身につけるためのWebアプリです。

## 収録内容

- 28単元・112小分類
- QA済み通常問題 2,280問
- 1日12問の自動出題
- 選択式、空所補充、語句並べ替え、誤文訂正
- 英文和訳、和文英訳、文構造判定、入試実戦
- 音声回答と文字入力の切替
- 当日間違い直し
- 翌日・3・7・14・30・60・90日の復習
- 7日以内2回／30日以内3回の誤答による苦手判定
- 理由4択、短い音声説明、類題
- 単元別進捗、苦手一覧、保護者向けCSV
- 学習記録のJSONバックアップ・復元
- 任意のGoogle Apps Script同期

## ファイル

- `index.html`：アプリ画面
- `styles.css`：画面デザイン
- `app.js`：学習・採点・復習エンジン
- `questions.js`：全2,280問
- `manifest.webmanifest` / `sw.js`：ホーム画面追加・オフライン対応
- `gas/`：任意のクラウド同期用Google Apps Script

## 使用方法

このフォルダの中身をHTTPS対応の静的Webホスティングへそのまま配置してください。
公開されたURLをiPhoneまたはiPadで開けば利用できます。

文字回答とタップ回答は通常のWebページとして動きます。音声認識、ホーム画面追加、オフラインキャッシュを安定して利用するには、HTTPSで公開してください。

## 学習記録

初期状態ではブラウザの端末内保存領域へ記録します。

「記録」画面から以下を実行できます。

- JSON形式でバックアップ
- バックアップから復元
- 保護者向けCSVの出力

端末変更やブラウザデータ削除に備え、定期的なバックアップを推奨します。

## 任意のGoogle同期

1. 新しいGoogleスプレッドシートを作成します。
2. そのスプレッドシートのApps Scriptへ`gas/Code.gs`を貼り付けます。
3. `setup()`を1回実行して`AppState`シートを作成します。
4. Webアプリとしてデプロイします。
5. 発行されたURLを、アプリの「設定」→「Google Apps Script同期URL」へ入力します。
6. 「クラウド保存」または「クラウド読込」を使用します。

状態データはセル上限を超えないよう、複数行へ分割して保存します。

## 採点について

- 選択式は正解番号で判定します。
- 文構造問題は選択した語句範囲を判定します。
- 英文回答は大文字小文字、句読点、一般的な短縮形を正規化します。
- 和訳は完全一致だけでなく文字列類似度も利用します。
- 「惜しい」は正解扱いにせず、翌日復習へ送ります。
- 音声認識失敗は文法上の誤答として記録しません。

## データ品質

問題データは次の検査を通過しています。

- 全2,280問の問題ID一意性
- 正解番号と選択肢の一致
- 空所と完成英文の一致
- 並べ替え語句と正解英文の一致
- 誤文と修正文の相違
- 和訳・英作文の回答方向
- 文構造のタップ対象
- 理由4択の正解
- 人物名と代名詞の一致
- 完全同一問題0件
